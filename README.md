# PHP CMS Blog

## Files

`admin.php` - Adding, editing and deleting posts.  
`delete.php` - Deletes posts from the `admin.php` page  
`edit.php` - Edit posts from the `admin.php` page  
`index.php` - See all posts  
`login.php` - Login to add, edit or delete posts  
`schema.sql` - DB Table creation
